Link in Bio - Simple (Style 4)

A simple template/theme for Link in Bio. Just a profile image, cover image, social medial icons, link buttons with description, and different backgrounds by TheNocturnalDevGypsy

Aside the basic profile and cover images, social media icons, link buttons with description and simple footer (also kept the shaking button JS animation to get user's attention to a specific link), Style 4 has a blurred fixed image background and container background color is semi-transparent (80%), and can be configured separately in CSS.
